30    /**
31    * A utility class that can be used to determine is a class is a syntactic
32    * direct subclass of owl:Thing. A class is considered NOT to be a syntactic
33    * direct subclass of owl:Thing if ANY of the following conditions apply:
34    * <ol>
35    * <li>It is equal to the left hand side of a subclass axiom, where the right
36    * hand side is a named class other than owl:Thing</li>
37    * <li>It is an operand in an equivalent class axiom where at least one of the
38    * other other operands is an intersection class that has a named operand other
39    * than the class in question. For example
40    * {@code EquivalentClasses(A, (B and prop some C))}</li>
41    * </ol>
42    * This functionality is provided because it is useful for displaying class
43    * hierarchies in editors and browsers. In these situations it is needed because
44    * not all "orphan" classes are asserted to be subclasses of owl:Thing. For
45    * example, if the only referencing axiom of class A was ObjectDomain(propP A)
46    * then A is a syntactic subclass of owl:Thing.
