39    /**
40    * A target method parameter that is annotated with this annotation allows to forward an intercepted method
41    * invocation to another instance. The instance to which a method call is forwarded must be of the most specific
42    * type that declares the intercepted method on the intercepted type.
43    * <p>&nbsp;</p>
44    * Unfortunately, before Java 8, the Java Class Library does not define any interface type which takes a single
45    * {@link java.lang.Object} type and returns another {@link java.lang.Object} type. For this reason, a
46    * {@link net.bytebuddy.implementation.bind.annotation.Pipe.Binder} needs to be installed explicitly
47    * and registered on a {@link net.bytebuddy.implementation.MethodDelegation}. The installed type is allowed to be an
48    * interface without any super types that declares a single method which maps an {@link java.lang.Object} type to
49    * a another {@link java.lang.Object} type as a result value. It is however not prohibited to use generics in the
50    * process. The following example demonstrates how the {@code @Pipe} annotation can be installed on a user type.
51    * As a preparation, one needs to define a type for which the {@code @Pipe} implements the forwarding behavior:
52    * <pre>
53    * interface Forwarder&lt;T, S&gt; {
54    * T forwardTo(S s);
55    * }
56    * </pre>
57    * Based on this type, one can now implement an interceptor:
58    * <pre>
59    * class Interceptor {
60    * private final Foo foo;
61    *
62    * public Interceptor(Foo foo) {
63    * this.foo = foo;
64    * }
65    *
66    * public String intercept(@Pipe Forwarder&lt;String, Foo&gt; forwarder) {
67    * return forwarder.forwardTo(foo);
68    * }
69    * }
70    * </pre>
71    * Using both of these types, one can now install the
72    * {@link net.bytebuddy.implementation.bind.annotation.Pipe.Binder} and register it on a
73    * {@link net.bytebuddy.implementation.MethodDelegation}:
74    * <pre>
75    * MethodDelegation
76    * .to(new Interceptor(new Foo()))
77    * .appendParameterBinder(Pipe.Binder.install(ForwardingType.class))
78    * </pre>
79    *
80    * @see net.bytebuddy.implementation.MethodDelegation
81    * @see net.bytebuddy.implementation.bind.annotation.TargetMethodAnnotationDrivenBinder
82    */
